# 🌍 RefuLearn — Donation Platform for Education

A modern, high-impact web platform designed to support Rohingya students through donations and storytelling.

This project focuses on delivering a **clean UI/UX, emotional storytelling, and a trust-driven experience** to encourage user engagement and donations.

---

## 🚀 Live Demo
https://refulearn.vercel.app/

---

## 🎯 Project Goal

The goal of this platform is to:
- Raise awareness about education challenges
- Provide a donation experience that feels **simple, emotional, and trustworthy**
- Showcase student stories in a modern and engaging way

---

## ⚙️ Tech Stack

### Frontend
- **Next.js (App Router)**
- **React**
- **TypeScript**

### UI / UX
- **Tailwind CSS**
- **Framer Motion (Animations)**
- **Lucide Icons & React Icons**
- **React Simple Typewriter**

---

## ✨ Features

### 🌐 Modern UI/UX
- Clean, minimal and responsive design
- Glassmorphism + gradient-based UI
- Mobile-first approach

### ⚡ Performance
- Fast navigation (SPA-like experience)
- Optimized rendering with `whileInView`
- Lightweight and scalable structure

### 📖 Storytelling
- Dynamic student pages (`/students/[id]`)
- Emotion-driven layout
- Image-focused design

### 💸 Donation Flow (UI)
- Preset donation amounts
- Custom donation input
- Frequency selection (one-time / monthly)

### 🔐 Trust Elements
- Transparency section
- Impact metrics
- Secure & global indicators
